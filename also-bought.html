Also Bought

Module for Magento 2​

Leave review to help in further development

  • Marketplace
  • Release Notes
  • Quality Report

Overview​

This module allows to add a block of products known as "Frequently Bought Together" or "Who Bought This Also Bought". These Also Bought products can be further displayed using a custom widget in a location of your choice or/and in Related, Up-Sell, Cross-Sell products.

This module will automatically create a list of products that were present in other customer orders, along with the currently viewed product. These products list is updated automatically according to the cron schedule you configure or manually in the Admin or using CLI command.

Tasks performed​

  • Collect products that were present in other customer orders along with the product being viewed in a separate table using:
    • Cron (automatically according to schedule).
    • Update Products button in Admin.
    • CLI command.
  • Show Also Bought products in:
    • custom VCT Also Bought widget(s).
    • Related, Up-Sell, Cross-Sell products.
  • Show Also Bought products in a widget(s) for:
    • Specific products manually selected.
    • All product types, Bundle, Configurable, Downloadable, Grouped, Simple, Virtual products.
  • Show or hide Also Bought products in Related, Up-Sell, Cross-Sell products.
  • Show random Also Bought products instead of last purchased products in a custom VCT Also Bought widget, Related, Up-Sell, Cross-Sell products.
  • Configure Also Bought widget:
    • Set title for widget(s).
    • Show or hide the following product card elements in Also Bought widget of your choice separately for each widget:
      • Product image, Price box.
      • Reviews summary (products rating and number of reviews).
      • Add To Cart, Add To Wish List, Add To Compare buttons.
      • Configurable products options in case of Visual Swatch or Text Swatch.
    • Set Also Bought widget location of your choice from the following containers:
      • After Page Header, After Page Header Top, Alert Urls.
      • Before Main Columns, Main Content Top, Main Content Container, Main Content Area, Main Content Aside, Main Content Bottom.
      • Mini-Cart Promotion Block.
      • Page Header, Page Header Container, Page Header Panel, Page Top, Page Bottom.
      • Product Auxiliary Info, Product Info Auxiliary Container, Product Social Links Container, Product View Extra Hint, Review Form Fields Before.
      • Sidebar Main, Sidebar Additional, Compare Link Wrapper.
      • Before Page Footer, Page Footer, Before Page Footer Container, Page Footer Container, CMS Footer Links.

Features​

  • Wide customization of the Also Bought product block by implementing it with a widget.
  • Tested and verified by Adobe Extension Quality Program.
  • Meets Magento Coding Standard.
  • Plugins (Interceptors) are used to prevent conflicts among modules.

Installation​

Use Composer to install the module or get the code for review:

  • Log in to your Marketplace account that purchased the module.
  • Add or update your Access Keys for Adobe Commerce Marketplace repository in auth.json using the following command:
composer config http-basic.repo.magento.com <Public Key> <Private Key>
# e.g. composer config http-basic.repo.magento.com 39b747b8ab1d624582bb3n1a09deb489 31b9fce4cb78f523fd34aa3abb90c89c

where <Public Key> and <Private Key> are your Access Keys.

  • Run the following commands:
composer require vct/alsobought # Install the module using Composer
bin/magento setup:upgrade --safe-mode=1 # Update the database schema and data
bin/magento setup:static-content:deploy --force # Deploy static view files
bin/magento setup:di:compile # Recompile the code only in production mode

Get your authentication keys and install an extension in the Adobe Commerce Documentation.

Configuration​

IMPORTANT

Flush Magento Cache in SYSTEM Tools Cache Management after configuration change to see the changes!

Clean and flush cache types in the Adobe Commerce Documentation.

Create VCT Also Bought widget(s)​

Use a custom VCT Also Bought widget to display Also Bought products: Content ELEMENTS Widgets Add Widget WIDGET SETTINGS Type VCT Also Bought.

TIP

Widget configs can be set separately for each widget.

Creating and managing widgets in the Adobe Commerce Documentation.

Update Products in widget​

IMPORTANT

Changes to Also Bought products will only be visible in the frontend after updating using:

  • Cron (scheduled tasks) automatically according to schedule in Stores SETTINGS Configuration ADVANCED System Cron (Scheduled Tasks) Cron configuration options for group: vct_alsobought (recommended way).
  • Update Products button in Admin in Stores SETTINGS Configuration VCT Also Bought Update.
  • bin/magento vct:alsobought:update command.

Configure vct_alsobought cron group​

Configure Cron configuration options for group: vct_alsobought in Stores SETTINGS Configuration ADVANCED System Cron (Scheduled Tasks) to set a schedule for Also Bought products updates.

INFO

By default, Also Bought products are updated with cron every 60 minutes.

Cron (scheduled tasks) in the Adobe Commerce Documentation.

Known issue​

ISSUE

Add To Cart button in products card can cause a JavaScript error in containers containing a form tag e.g. Review Form Fields Before container.

This is happening because the button itself already contains a form tag.

FIX

Set Content ELEMENTS Widgets [VCT Also Bought] WIDGET Widget Options Show Add To Cart to No to disable Add To Cart button in containers with a form tag.
